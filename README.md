# 🔗 URL Shortener

A simple and efficient URL Shortener built with modern web technologies. It allows users to convert long URLs into short, easy-to-share links — just like Bitly or TinyURL!

---

## 🛠️ Tech Stack

**Frontend:**
- React 
- Tailwind CSS

**Backend:**
- JavaScript
- Supabase
